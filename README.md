# Loja-de-Tintas-Online: 🎨 TINTEL.ONLINE

Sistema web/e-commerce desenvolvido para modernizar a experiência de compra de tintas, materiais de pintura e produtos relacionados, oferecendo praticidade, personalização e uma navegação intuitiva.

O projeto foi desenvolvido com foco em experiência do usuário (UX), identidade visual moderna e processos digitais que aproximam clientes e lojas do segmento de tintas.

---

## 📌 Sobre o Projeto

O TINTEL.ONLINE surgiu com o objetivo de solucionar dificuldades comuns encontradas no processo tradicional de compra de tintas e produtos relacionados, como:

* Dificuldade na escolha de cores;
* Falta de praticidade durante a compra;
* Pouca personalização na experiência do usuário;
* Ausência de ferramentas digitais intuitivas;
* Dificuldade para comparar produtos.

A plataforma busca proporcionar uma experiência moderna, simples e eficiente através de recursos digitais e um ambiente visual profissional.

---

## 🎯 Objetivo Geral

Desenvolver uma plataforma web/e-commerce moderna para o segmento de tintas, unindo:

* Experiência do usuário (UX);
* Design moderno e intuitivo;
* Facilidade na navegação;
* Conversão de vendas;
* Fortalecimento da marca.

## ⚙ Como executar o projeto

### Clonar repositório

```bash
git clone URL_DO_REPOSITORIO
```

### Entrar na pasta

```bash
cd TINTEL.ONLINE
```

### Instalar dependências

```bash
npm install
```

### Executar projeto

```bash
npm run dev
```

O projeto estará disponível em:

```bash
http://localhost:5173
```
